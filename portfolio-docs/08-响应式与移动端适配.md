# Step 8: 响应式与移动端适配

## 8.1 功能概述

将所有页面适配到移动端、平板、PC端，实现底部Tab导航和触摸交互优化。

## 8.2 断点定义

| 断点 | 范围 | 设备 |
|-----|------|------|
| Mobile | < 640px | 手机 |
| Tablet | 640px - 1024px | 平板 |
| Desktop | > 1024px | PC |

## 8.3 移动端导航

### 底部Tab导航

| 入口 | 图标 | 路由 | 说明 |
|-----|------|------|------|
| 首页 | Home | `/` | 个人简介 |
| 作品 | Briefcase | `/projects` | 作品列表 |
| Agent | MessageCircle | `/agent` | AI助手 |
| 预约 | Calendar | `/booking` | 预约表单 |

**特性：**
- 固定在视口底部
- 当前选中项高亮（Accent色 #0066FF）
- 点击切换，无页面刷新（SPA）
- 支持手势滑动切换（可选）

**布局：**
```
┌─────────────────────────────┐
│                             │
│       页面内容区域           │
│                             │
│                             │
├─────────────────────────────┤
│  🏠       💼       💬       📅      │
│  Home    Works    Agent    Book    │
└─────────────────────────────┘
```

## 8.6 响应式CSS

```css
/* 移动端基础样式（Mobile First） */
.container { padding: 16px; }
.card { margin-bottom: 12px; }
.works-grid { grid-template-columns: 1fr; }

/* 平板适配 */
@media (min-width: 640px) {
  .container { padding: 24px; }
  .works-grid { grid-template-columns: repeat(2, 1fr); }
}

/* PC适配 */
@media (min-width: 1024px) {
  .container { max-width: 1200px; margin: 0 auto; padding: 32px; }
  .works-grid { grid-template-columns: repeat(3, 1fr); }
  .home-layout { display: grid; grid-template-columns: 1fr 2fr; }
}
```

## 8.7 性能优化

- 图片懒加载（Intersection Observer）
- 路由懒加载（React.lazy）
- 关键CSS内联
- 图片使用WebP格式
- 字体最小16px（避免iOS缩放）
