# Step 4: 作品展示

## 4.1 功能概述

展示企业项目和个人项目，支持分类筛选、详情查看。管理后台可CRUD作品。

## 4.2 数据库表

```sql
CREATE TABLE projects (
    id SERIAL PRIMARY KEY,
    name VARCHAR(200) NOT NULL,
    type VARCHAR(20) CHECK (type IN ('enterprise', 'personal')),
    start_date DATE,
    end_date DATE,
    summary TEXT,
    description TEXT,
    cover_image VARCHAR(500),
    images TEXT[],
    github_url VARCHAR(500),
    demo_url VARCHAR(500),
    tags VARCHAR(50)[],
    sort_order INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 索引
CREATE INDEX idx_projects_type ON projects(type);
CREATE INDEX idx_projects_sort_order ON projects(sort_order);
CREATE INDEX idx_projects_created_at ON projects(created_at DESC);
```

## 4.3 API设计

```
GET    /api/projects                # 获取作品列表（公开，支持分页）
GET    /api/projects/:id            # 获取作品详情（公开）
POST   /api/projects                # 创建作品（需登录）
PUT    /api/projects/:id            # 更新作品（需登录）
DELETE /api/projects/:id            # 删除作品（需登录）
```

**GET /api/projects 请求参数：**
```
?type=enterprise&page=1&page_size=20&sort=sort_order&order=desc
```

## 4.5 交互要求

- Tab切换：平滑过渡动画
- 卡片悬停：上浮阴影 + "查看详情"遮罩
- 图片轮播：支持手势滑动（移动端）
- 删除确认：二次确认弹窗
- 表单校验：实时校验 + 提交校验

## 4.7 缓存策略

| Key | TTL | 说明 |
|-----|-----|------|
| `projects:list:enterprise` | 30分钟 | 企业项目列表 |
| `projects:list:personal` | 30分钟 | 个人项目列表 |
| `projects:detail:{id}` | 30分钟 | 作品详情 |
