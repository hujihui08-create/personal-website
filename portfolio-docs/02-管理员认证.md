# Step 2: 管理员认证

## 2.1 功能概述

管理员通过密码登录后台，JWT Token认证保护管理接口。

## 2.2 数据库表

```sql

    id SERIAL PRIMARY KEY,
    password_hash VARCHAR(255) NOT NULL,
    notification_email VARCHAR(255),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
```

## 2.3 API设计

### POST /api/auth/login

**请求：**
```json
{
  "password": "admin123"
}
```

**成功响应（200）：**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "expires_at": "2026-05-14T10:30:00Z"
}
```

**错误响应（401）：**
```json
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "密码错误"
  }
}
```

### POST /api/auth/logout

**请求头：** `Authorization: Bearer <token>`

**响应（200）：**
```json
{
  "message": "已登出"
}
```

### GET /api/auth/me

**请求头：** `Authorization: Bearer <token>`

**响应（200）：**
```json
{
  "id": 1,
  "notification_email": "admin@example.com",
  "created_at": "2026-01-01T00:00:00Z"
}
```

## 2.4 后端实现

### JWT中间件

```go
// internal/middleware/auth.go
func AuthMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        tokenString := c.GetHeader("Authorization")
        if tokenString == "" {
            c.JSON(401, gin.H{"error": gin.H{
                "code": "UNAUTHORIZED",
                "message": "缺少认证Token",
            }})
            c.Abort()
            return
        }

        // 去除 "Bearer " 前缀
        tokenString = strings.TrimPrefix(tokenString, "Bearer ")

        token, err := jwt.Parse(tokenString, func(token *jwt.Token) (interface{}, error) {
            return []byte(config.JWTSecret), nil
        })

        if err != nil || !token.Valid {
            c.JSON(401, gin.H{"error": gin.H{
                "code": "UNAUTHORIZED",
                "message": "Token无效或已过期",
            }})
            c.Abort()
            return
        }

        c.Set("admin_id", token.Claims.(jwt.MapClaims)["admin_id"])
        c.Next()
    }
}
```

### 登录限流

```go
// 登录失败计数（Redis）
func LoginRateLimit() gin.HandlerFunc {
    return func(c *gin.Context) {
        ip := c.ClientIP()
        key := fmt.Sprintf("login:fail:%s", ip)

        count, _ := redis.Get(ctx, key).Int()
        if count >= 5 {
            c.JSON(429, gin.H{"error": gin.H{
                "code": "RATE_LIMITED",
                "message": "登录失败次数过多，请15分钟后再试",
            }})
            c.Abort()
            return
        }
        c.Next()
    }
}
```

### 请求拦截器

```typescript
// api/client.ts
const apiClient = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
});

apiClient.interceptors.request.use((config) => {
  const token = useAuthStore.getState().token;
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

apiClient.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      useAuthStore.getState().logout();
      window.location.href = '/admin/login';
    }
    return Promise.reject(error);
  }
);
```

## 2.6 安全要求

| 项 | 措施 |
|----|------|
| 密码加密 | bcrypt，工作因子12 |
| JWT有效期 | 7天 |
| Token存储 | localStorage（HttpOnly Cookie需后端配合） |
| 登录限流 | 5次失败锁定15分钟 |
| 密码传输 | 仅HTTPS |
