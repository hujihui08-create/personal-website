项目详情

该项目为某大型产业园区量身定制，涵盖物业管理、能耗监控、访客预约、停车管理等子系统。采用微服务架构，前端使用 React + TypeScript 构建，后端使用 Go + Gin 框架。 技术亮点： • 基于 MQTT 协议的 IoT 设备实时数据采集与可视化 • 自研规则引擎实现智能告警与自动化响应 • 多租户架构支持园区内多家企业独立管理 • 大屏数据看板支持 7x24 小时实时监控
